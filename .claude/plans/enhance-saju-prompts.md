# 사주 인사이트 시스템 프롬프트 보강

## 개요
사용자가 Claude 웹 Opus 세션에서 만든 상세 사주 분석 프롬프트를 기존 인사이트 시스템(Sonnet 에이전트 + Opus 분석 task)에 반영.
기존에 이미 있는 정보는 패스하고, 사주 분석 품질에 직접 영향을 주는 핵심만 추가.

## 변경 대상 4곳

### 1. DB: saju_profiles.profile_summary 확장
현재 profile_summary에 없는 핵심 구조 추가. weekly-fortune과 weekly-saju-review 모두 이 필드를 읽으므로 여기에 넣으면 양쪽 다 혜택.

**추가 내용:**
- 편인(戌) ×2 심층: 천간에 없음(겉으로 안 보이는 편인), 정인(正印) 원국 부재, "이해해야 안전하다" 생존 회로
- 양간(庚) 여성 특성
- 금생수 패턴 핵심: 에너지를 주고 존재가치를 돌려받는 구조 → 2026.02 방향 전환 완료
- 경금련정 다이어트 원칙: "적당한 온도로 오래" — 월 0.5\~0.7kg
- 을사년(2025) 핵심 전환: 생존 모드 → 존재 모드, 불안회로 절단
- 배우자 핵심: 丁未 일주 최적, 丁卯/壬午/丁亥/壬寅 차선. 2027 정미년 결혼 최적

### 2. src/agents/insight/prompt.ts — Sonnet 에이전트 프롬프트 보강
Sonnet은 직접 사주 해석을 안 하고 fortune_analyses 기반으로 말하지만, 사용자 맥락을 더 알면 일기 응답+일운 전달의 질이 올라감.

**추가 내용 (프롬프트 하단, fortuneContext 앞에):**
```
## 사용자 핵심 맥락 (사주 기반)

### 편인(戌) ×2 심층
- 천간에 없음 = 겉으로 안 보이는 편인. "괜찮은 척" 완벽 연기.
- 정인(正印) 원국에 전혀 없음 — 안전한 기반 없이 편인만 작동. 탐구가 "호기심"보다 "생존 전략"에 가깝게 기울어지기 쉬움.
- "이해해야 안전하다" 생존 회로 (7세 형성). 과열 시 "울면서도 질문을 멈출 수 없는" 패턴.
- ⚠️ 편인 과열 신호 감지 시 (멈출 수 없는 탐구, 공부로 감정 회피) 부드럽게 멈추도록 안내.
- 명리학은 "정답지"가 아니라 "지도" — "어차피 흐름이니까"가 체념이면 짚어줘.

### 금생수 패턴
- 30년간 "나보다 못한 사람에게 베풀어주는" 패턴 반복. 2026.02 경인월에 방향 전환 완료.
- "이젠 내가 받겠다" — 연애, 직장, 모든 관계에 적용.
- 일기에서 금생수 재발현 신호 감지 시 짚어줘.

### 현재 대운·세운 핵심
- 갑자대운(2020~2029): 편재+상관. "깨는" 대운. 엄마(甲子 일주)와 같은 글자의 대운.
- 병오년(2026): 편관 세운. 경금련정 단련의 해. 화극금 주의(피부, 수면).
- 다음 대운 계해(2030~2039)가 인생의 황금기 — 지금은 그 준비기.

### AI 주의사항
- 시지 亥 = 식신(壬水). 상관 아님. 대운 子 = 상관(癸水). 이 구분 유지.
- 丑 본기 = 己土 = 정인. 편인 아님!
- 편인 과다 = 괴로운 정지(분석 마비). 정인 과다 = 편안한 정지(게으름, 식욕 증가). 혼동 금지.
```

**변경 위치**: `buildInsightSystemPrompt()` 함수 내, `${fortuneContext}` 앞에 삽입.
이 내용은 코드에 하드코딩하는 게 아니라, life_themes/saju_patterns처럼 동적으로 넣을 수도 있지만, 이 정보는 변하지 않는 원국 기반이라 프롬프트에 직접 넣는 게 맞음.

원본(엄마 甲子 일주) → 복사본1(전 회사 대표 壬辰) → 복사본2(전남친 癸未).

## 패스하는 항목 (이미 있거나 불필요)

- 연애 이력 상세 일지 기록 → 너무 길고, 핵심 패턴만 배우자 분석에 반영
- 인생 최고/최악 시기 랭킹 → fortune이 대운 흐름으로 자체 판단 가능
- 고양이 물상 → 사주 분석 핵심 아님
